# todos-uni-app
基于uni-app的一款多端项目。  
Todo —— 专注要做的事。
## 功能需求
- 已完成功能
  - 添加待做项
  - 完成待做项（点击未完成项）
  - 删除已完成项（点击已完成项）
  - 时间戳格式化（YYYY-m/mm-dd hh:mm:ss）
  - 多端 其他未进行测试
    - 微信小程序
	- H5
- 待完成功能
  - 优化todo列表
  - 使用vuex
  - 使用本地存储
  - ……
